## 概览

提供一套标准的OAuth2流程, 可以支持access_token的授权/生成/生命周期管理等, 并且包含scope由于被调用方做一些权限控制且统一纳管.



## 功能特性

- 提供 Client 统一管理服务
- 管理 Client 访问网关的权限
- 支持 OAuth2: authorization code
- 提供 access_token 生命周期管理
